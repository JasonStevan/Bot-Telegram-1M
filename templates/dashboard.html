Manual Rápido

  1. Vá para Configurações
  2. Insira o token do seu bot obtido do @BotFather no Telegram
  3. Insira o ID do grupo do Telegram onde o bot atuará
  4. Defina o intervalo desejado (em minutos) entre postagens promocionais
  5. Salve as configurações e depois ative o bot no Dashboard

  1. Vá para Posts Promocionais
  2. Clique em "Novo Post" para criar um novo post promocional
  3. Adicione título, conteúdo, uma URL de imagem (opcional) e um link externo (opcional)
  4. Salve o post - ele será enviado automaticamente na sequência quando o bot estiver ativo
  5. Você pode editar, excluir ou testar o envio de qualquer post a qualquer momento

  1. Vá para Mensagens de Boas-vindas
  2. Personalize a mensagem que será enviada aos novos membros do grupo
  3. Use as variáveis {first_name}, {last_name} e {username} para personalizar a mensagem
  4. Ative ou desative o envio de mensagens de boas-vindas com a caixa de seleção
  5. Clique em "Enviar Teste" para ver como sua mensagem aparecerá no grupo
